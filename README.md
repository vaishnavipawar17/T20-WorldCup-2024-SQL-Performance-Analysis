# T20-WorldCup-2024-SQL-Performance-Analysis
SQL project analyzing team and player performance in T20 World Cup 2024 using queries, joins, subqueries and aggregations.

# Project Overview
This project analyzes team and player performance in the T20 World Cup 2024 using SQL.  
It includes various queries using joins, subqueries, aggregate functions, and grouping to extract meaningful insights from cricket match data.

---

# Tools & Technologies Used
- MySQL
- SQL
- Joins
- Subqueries
- Aggregate Functions
- Group By & Having
- Data Analysis Concepts

---

# Key Analysis Performed

1. Team performance summary (Matches Played, Wins, Win %)
2. Top 5 highest run scorers
3. Players with most wickets
4. Matches won by highest margin
5. Team win comparison
6. Performance trends
